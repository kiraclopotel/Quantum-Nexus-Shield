## Usage

### Basic Usage


from quantum_nexus_shield import QuantumStackEncryption

# Initialize the encryption system
encryption = QuantumStackEncryption()

# Encrypt a message
message = b"Your secret message"
success, entropy = encryption.add_message(message)

# Get encryption details
if success:
    print(f"Message encrypted with entropy: {entropy}")
